# Equipment Tracker
Simple equipment management app.

## Tech Used
- React
- Node.js
- Express
- MySQL

## Features
- View all equipment in a table
- Add new equipment
- Edit existing equipment
- Delete equipment
- Search equipment by name
- Sort equipment by name (A–Z / Z–A)


## How to Run
1. Create MySQL database and table
2. Run backend using `node server.js`
3. Run frontend using `npm start`

## Assumptions
- No login required
- Simple UI only
